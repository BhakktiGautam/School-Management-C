🏫 School Management System in C


A console-based School Management System built using C programming language.
This project demonstrates how core programming concepts like structures, arrays, loops, and modular functions can be used to simulate a basic real-world management system.

---

📌 Project Overview

The system allows management of:

- 👨‍🎓 Students
- 👩‍🏫 Teachers
- 🏫 Classes
- 📊 Student Results

It is a menu-driven application where the user can perform various operations like adding records, searching, updating, calculating results, and displaying information.

---

🚀 Features

👨‍🎓 Student Management

- Add new student
- Remove student
- Update student details
- Search student by Roll Number
- Display all students

👩‍🏫 Teacher Management

- Add teacher
- Remove teacher
- Update teacher details
- Search teacher
- Display teacher records

🏫 Class Management

- Create class
- Assign class teacher
- Display students of a class
- Calculate average marks of class

📊 Result Section

- Calculate total marks
- Calculate percentage
- Assign grade
- Display class topper
